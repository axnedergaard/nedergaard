# Neuropipes
An efficient and extensible real-time EEG signal processing framework written in C

Dependencies: mcrypt, hidapi-libusb, fftw3, uthash

Supported EEG systems: EMOTIV Epoc, OpenEEG (Raspberry Pi serial port)
